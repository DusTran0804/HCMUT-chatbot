# 🤖 AI RAG Chatbot – HCMUT Assistant

An intelligent chatbot designed to answer questions about **Ho Chi Minh City University of Technology (HCMUT – VNU-HCM)** using **Retrieval-Augmented Generation (RAG)**.

The system retrieves relevant information from a **vectorized knowledge base** and generates context-aware responses using an AI model.

---

# ⚙️ Key Features

- 📄 **Document Chunking** for better context handling  
- 🧠 **Vector Embedding** for semantic search  
- 🔎 **Similarity-based Retrieval**  
- 🤖 **LLM-based Answer Generation**  
- ⚡ Lightweight implementation
